Piazza Classifier

Code available upon request

Project Spec

I created a program that automatically identified the subject of class Piazza posts using natural language processing and machine learning techniques.

I trained the program using a trainng set, developed a simplified version of a Multivariate Bernoulli Naive Bayes Classifier, and tested my program using a validation set, to successfully classify new Piazza posts.

Completed as part of a class project.
