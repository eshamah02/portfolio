A3: Aggregating AWS (Infrastructure) Attributes

I developed an end-to-end generation and serving procedure to aggregate and display AWS infrastructure attributes to deploy centralized units. I used Lambda functions and Step Functions to aggregate data from DynamoDB tables, EC2 instances, and VPCs to fill up an Aurora Serverless database. I implemented dynamic updates for the Aurora database by using DynamoDB streams and trigger functionality. I hosted a REST API on API Gateway using Flask to deliver a user-friendly and dynamic front-end experience with user authentication.
