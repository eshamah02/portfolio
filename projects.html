a3: aggregating aws (infrastructure) attributes

i developed an end-to-end generation and serving procedure to aggregate and display aws infrastructure attributes to deploy centralized units. i used Lambda functions and Step Functions to aggregate data from DynamoDB tables, EC2 instances, and VPCs to fill up an aurora serverless database. i implemented dynamic updates for the aurora database by using DynamoDB streams and triggers. i hosted a REST API on Flask and API Gateway to deliver a user-friendly and dynamic front-end experience with authentication.

tech stack: AWS Cloud services (Lambda, Step Functions, DynamoDB, EC2, VPC, RDS, IAM, etc.), Python, Flask

chess AI

iteratively developed a chess AI program. started with minimax alpha beta pruning and a simple heuristic using FEN representation. then, created and trained a simple deep neural network model using TensorFlow using ReLu activation and sigmoid activation.

tech stack: Python, TensorFlow ML library
